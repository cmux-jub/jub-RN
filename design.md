# Design Guide

이 파일은 `jub` RN UI 구현의 선행 확인 문서다.

- UI 코드 작성 전 먼저 이 파일을 확인한다
- 색상 토큰을 새로 만들기 전, 여기 정의된 팔레트를 우선 사용한다
