# jub-RN

이 프로젝트는 Google Play Store 배포를 목표로 하는 제품 빌드가 아니라, `.apk` 형태로 시연하고 공유하기 위한 React Native 프로젝트입니다.

`jub`의 React Native 구현 하네스 문서는 [PROJECT_HARNESS.md](./PROJECT_HARNESS.md) 에 정리되어 있습니다.

API 기준 문서는 [API_SPEC.md](./API_SPEC.md) 에 정리되어 있으며, 이후 RN 네트워크 구현은 이 문서를 기준으로 맞춥니다.

디자인 기준 문서는 [design.md](./design.md) 에 정리되어 있으며, 이후 UI 코드를 작성하기 전 반드시 먼저 확인합니다.
